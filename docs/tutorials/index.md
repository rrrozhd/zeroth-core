# Tutorials

Tutorials are **learning-oriented**: they walk you step-by-step through a concrete task so you can build working knowledge of Zeroth in a single sitting. If this is your first visit, start with Getting Started and then tackle the Governance Walkthrough to see Zeroth's differentiators in action.

- [Getting Started](getting-started/index.md) — install, build your first graph, and run it as a service with an approval gate.
- [Governance Walkthrough](governance-walkthrough.md) — exercise the approval gate, auditor, and policy block end-to-end.
