**Step 1: Create `src/zeroth/storage/database.py`** (per D-07) with:

```python
from __future__ import annotations
from typing import Any, Protocol, runtime_checkable
from contextlib import asynccontextmanager
from collections.abc import AsyncIterator

@runtime_checkable
class AsyncConnection(Protocol):
    """Abstraction over a database connection within a transaction."""
    async def execute(self, sql: str, params: tuple[Any, ...] = ()) -> None: ...
    async def fetch_one(self, sql: str, params: tuple[Any, ...] = ()) -> dict[str, Any] | None: ...
    async def fetch_all(self, sql: str, params: tuple[Any, ...] = ()) -> list[dict[str, Any]]: ...
    async def execute_script(self, sql: str) -> None: ...

@runtime_checkable
class AsyncDatabase(Protocol):
    """Abstract async database interface for all repositories."""
    @asynccontextmanager
    async def transaction(self) -> AsyncIterator[AsyncConnection]: ...
    async def close(self) -> None: ...
```

**Step 2: Create `src/zeroth/storage/async_sqlite.py`** (per D-11) with:

- `_convert_row(row) -> dict[str, Any]` helper that converts sqlite3.Row to dict
- `class AsyncSQLiteConnection` implementing `AsyncConnection`:
  - Constructor takes `aiosqlite.Connection`
  - `execute()`: runs `await self._conn.execute(sql, params)`
  - `fetch_one()`: runs query, returns dict or None via `_convert_row`
  - `fetch_all()`: runs query, returns list of dicts via `_convert_row`
  - `execute_script()`: runs `await self._conn.executescript(sql)`
- `class AsyncSQLiteDatabase` implementing `AsyncDatabase`:
  - Constructor takes `path: str`, `encryption_key: str | bytes | None = None`
  - Stores `self.path`, creates `EncryptedField` if key provided (reuse from `sqlite.py`)
  - `transaction()`: async context manager that opens aiosqlite connection, sets `row_factory = aiosqlite.Row`, executes PRAGMAs (`foreign_keys = ON`, `journal_mode = WAL`, `synchronous = NORMAL`), yields `AsyncSQLiteConnection`, commits on success, rollbacks on error, closes connection
  - `close()`: no-op (connections are per-transaction)

**Step 3: Create `src/zeroth/storage/async_postgres.py`** (per D-10) with:

- `import re; _PLACEHOLDER_RE = re.compile(r"\?")` for `?` to `%s` conversion
- `def _sqlite_to_psycopg(sql: str) -> str` that converts `?` placeholders to `%s`
- `class PostgresConnection` implementing `AsyncConnection`:
  - Constructor takes `psycopg.AsyncConnection`
  - `execute()`: converts placeholders, runs `await self._conn.execute(converted_sql, params)`
  - `fetch_one()`: converts placeholders, runs query with `cursor.fetchone()`, returns dict via `dict(zip(col_names, row))` or None
  - `fetch_all()`: converts placeholders, runs query, returns list of dicts
  - `execute_script()`: splits on `;`, executes each statement (skip empty)
- `class AsyncPostgresDatabase` implementing `AsyncDatabase`:
  - Constructor takes `pool: AsyncConnectionPool`
  - `@classmethod async def create(cls, dsn: str, *, min_size: int = 2, max_size: int = 10)`: creates `AsyncConnectionPool(dsn, min_size=min_size, max_size=max_size, open=False)`, `await pool.open()`, returns `cls(pool)`
  - `transaction()`: async context manager using `async with self._pool.connection() as conn: async with conn.transaction(): yield PostgresConnection(conn)`
  - `close()`: `await self._pool.close()`

**Step 4: Create `src/zeroth/storage/factory.py`** (per D-12) with:

```python
from zeroth.config.settings import ZerothSettings
from zeroth.storage.database import AsyncDatabase

async def create_database(settings: ZerothSettings) -> AsyncDatabase:
    if settings.database.backend == "postgres":
        from zeroth.storage.async_postgres import AsyncPostgresDatabase
        dsn = settings.database.postgres_dsn.get_secret_value()
        return await AsyncPostgresDatabase.create(
            dsn,
            min_size=settings.database.postgres_pool_min,
            max_size=settings.database.postgres_pool_max,
        )
    else:
        from zeroth.storage.async_sqlite import AsyncSQLiteDatabase
        return AsyncSQLiteDatabase(
            path=settings.database.sqlite_path,
            encryption_key=(
                settings.database.encryption_key.get_secret_value()
                if settings.database.encryption_key else None
            ),
        )
```
