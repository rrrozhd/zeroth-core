- `create_mirror`

- every stash from `git stash list` and create `archive/stash-<index>` branches

Use archive root `$HOME/archives/zeroth-monolith`. Create:
- tarball path `$HOME/archives/zeroth-monolith/zeroth-monolith-2026-04-10.tar.gz`
- bare mirror path `$HOME/archives/zeroth-monolith/zeroth-monolith.git`

`write_preflight_report` must write `.planning/phases/27-ship-zeroth-as-pip-installable-library-zeroth-core/artifacts/archive-preflight.txt` with exact `key=value` lines for:
- `tarball_path=...`
- `tarball_sha256=...`

- `mirror_ref_count=...`
