# Phase 22 -- UI Design Contract

> Visual and interaction contract for the Canvas Foundation & Dev Infrastructure phase. Generated by gsd-ui-researcher, verified by gsd-ui-checker.

---

## Design System

| Property | Value |
|----------|-------|
| Tool | none (Vue 3, not React -- shadcn not applicable) |
| Preset | not applicable |
| Component library | Vue Flow (@vue-flow/core) for canvas, custom Vue components for shell |

## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline padding, handle offsets |
| sm | 8px | Compact element spacing, node meta gaps |
| md | 16px | Default element spacing, panel padding base, canvas controls gap |
| lg | 24px | Section padding, canvas dot grid size, rail section gaps |
| xl | 32px | Layout gaps between panels |
| 2xl | 48px | Major section breaks, canvas minimap offset from edge |
| 3xl | 64px | Page-level spacing (header height region) |

Exceptions: Node handles use 10px diameter (not on 4px grid) for touch target precision. Canvas control buttons use 28px (7x4) for compact toolbar fit.

**Source:** Mockup CSS measurements normalized to 4px multiples. Canvas dot grid uses 24px (lg).

## Glassmorphic Surface Contract

All panels use a consistent frosted-glass treatment:

| Property | Value |
|----------|-------|
| Background | `rgba(255, 255, 255, 0.28)` |
| Border | `1px solid rgba(118, 182, 205, 0.3)` |
| Border radius | 12px (panels), 8px (controls, inputs), 6px (inner elements) |
| Inner highlight | `inset 0 1px 0 rgba(255, 255, 255, 0.6)` |
| Shadow | `0 18px 48px rgba(117, 160, 189, 0.08)` |
| Backdrop filter | `blur(20px)` |

Node cards use a gradient variant:
- Background: `linear-gradient(180deg, rgba(255, 255, 255, 0.38), rgba(236, 250, 255, 0.18))`
- Border radius: 12px
- Shadow: `0 16px 28px rgba(115, 164, 193, 0.06)`

**Source:** Mockup CSS `.panel` and `.graph-node` classes.

## Layout Contract

### Three-Panel Shell

| Region | Width | Behavior |
|--------|-------|----------|
| Header | 100%, min-height 76px | Fixed top, glassmorphic panel, contains title + mode switch + actions |
| Workflow rail | 304px (19rem) fixed | Left sidebar, collapsible via toggle button |
| Canvas | Flex 1, min-width 0, min-height 550px | Center workspace, pannable/zoomable |
| Inspector | 0px default (hidden in Phase 22) | Right sidebar, shown on node selection (Phase 23) |

Gap between body panels: 16px (1rem).
Shell min-width: 900px.
Shell padding: 18px.

### Mode Switch

Positioned: absolute, bottom of header, centered horizontally, translated 50% below header bottom.
Tabs: "Editor", "Executions" (Phase 22 renders Editor only as active).

### Canvas Controls

Position: absolute, bottom-left of canvas surface (bottom: 16px, left: 16px).
Buttons: Fit view, Zoom in, Zoom out, Tidy up, divider, Add node.
Button size: 28px x 28px, border-radius 5px.

### Minimap

Position: absolute, bottom-left of canvas (bottom: 72px, left: 16px).
Size: 110px x 72px, border-radius 6px.
Contains: proportional node rectangles, edge lines, viewport indicator.

**Source:** Mockup HTML structure and CSS layout declarations, CONTEXT.md D-07 (Phase 22 is visual placement only).

## Interaction Contract

### Canvas Interactions (Phase 22 scope)

| Interaction | Behavior |
|-------------|----------|
| Pan | Click + drag on empty canvas area (cursor: grab/grabbing) |
| Zoom | Scroll wheel or pinch gesture |
| Fit to view | Canvas control button, fits all nodes in viewport |
| Node drag | Click + drag on node card to reposition |
| Edge draw | Click + drag from output handle to input handle (cursor: crosshair on output handle) |
| Node select | Click on node card, shows selected border state |
| Multi-select | Not in Phase 22 scope |
| Delete | Via hover toolbar delete button on individual nodes |

### Hover Behaviors

| Element | Hover Effect |
|---------|-------------|
| Node card | Dark toolbar appears above with Run, Disable, Delete buttons |
| Node handle | Scale 1.5x, border color intensifies to full accent |
| Workflow row | Cyan tint background, action dots appear |
| Folder header | Cyan tint background, action buttons appear |
| Canvas control button | Accent tint background, text darkens |
| Mode switch tab | Text darkens |

### Transitions

All interactive state transitions use 120ms ease timing. Exceptions:
- Node toolbar appear: 100ms ease
- Handle hover scale: 150ms ease
- Running state pulse: 2s ease-in-out infinite

**Source:** Mockup CSS transition and animation declarations.
